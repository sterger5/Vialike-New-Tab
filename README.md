# Vialike New Tab

一个将 Chrome 新标签页替换为类似 Via 浏览器主页面的扩展，支持丰富的定制化选项。

## 安装方法

### 方法一：从 GitHub 下载安装

#### 第一步：下载项目

1. 访问本项目的 GitHub 仓库页面
2. 点击绿色的 "Code" 按钮
3. 选择 "Download ZIP" 下载项目压缩包
4. 解压下载的 ZIP 文件到任意文件夹

#### 第二步：在 Chrome 中安装扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并按回车
3. 打开页面右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"** 按钮
5. 在弹出的文件选择窗口中，选择解压后的项目文件夹
6. 安装成功后，扩展列表中会显示 "Vialike New Tab"

#### 第三步：开始使用

- 打开一个新的标签页，即可看到新的主页界面
- 点击页面右下角的齿轮图标打开设置面板

## 功能介绍

### 主界面功能

- **书签栏**：自动读取 Chrome 书签，显示在页面顶部
- **Logo**：可自定义的文字标识
- **搜索框**：支持网址跳转和搜索引擎搜索
- **设置按钮**：右下角齿轮图标，点击打开设置面板

### 搜索功能

- 输入网址（如 `github.com`）可直接跳转到该网站
- 输入文字后按回车或点击搜索按钮，使用选定的搜索引擎进行搜索
- 打开新标签页时自动聚焦到搜索框，可直接输入

### 快捷切换搜索引擎

- 右键点击搜索框右侧的放大镜图标
- 在弹出的列表中选择想要使用的搜索引擎

## 设置项说明

设置面板分为五个类别：**Logo**、**搜索框**、**背景**、**搜索引擎**、**高级**

### 一、Logo 设置

| 设置项 | 说明 | 范围/选项 |
|--------|------|-----------|
| Logo文字 | 显示在页面上方的标识文字 | 任意文字，默认为 "Vialike" |
| 字体大小 | Logo文字的大小 | 24px - 120px |
| 字体风格 | Logo使用的字体 | 默认、Arial、Times New Roman、Georgia、Courier New、Verdana、Impact、手写体、艺术体 |
| 字体样式 | 文字的粗细和倾斜 | 粗体(B)、斜体(I) |

### 二、搜索框设置

| 设置项 | 说明 | 范围/选项 |
|--------|------|-----------|
| 搜索框样式 | 搜索框的外观形状 | 矩形、线条 |
| 圆角值 | 搜索框的圆角程度（仅矩形样式） | 0px - 50px |
| 不透明度 | 搜索框的透明程度 | 0% - 100% |
| 描边粗细 | 搜索框边框的粗细 | 0px - 5px |
| 描边不透明度 | 边框的透明程度 | 0% - 100% |
| 显示提示文字 | 是否显示 "搜索或输入网址" 提示 | 开/关 |
| 文字颜色 | 搜索框内文字的颜色 | 颜色选择器 |

### 三、背景设置

| 设置项 | 说明 | 范围/选项 |
|--------|------|-----------|
| 背景图片 | 自定义背景图片 | 选择本地图片或清除 |
| 主题色 | 页面的明暗主题 | 亮色、暗色 |
| 背景不透明度 | 背景图片的透明程度 | 0% - 100% |

### 四、搜索引擎设置

| 设置项 | 说明 | 选项 |
|--------|------|------|
| 默认搜索引擎 | 搜索时使用的引擎 | Google、Bing、百度、DuckDuckGo、搜狗 |
| 快捷切换 | 右键点击放大镜图标快速切换引擎 | 提示说明 |

### 五、高级设置

| 设置项 | 说明 | 范围/选项 |
|--------|------|-----------|
| 书签栏 | 是否显示顶部书签栏 | 开/关 |
| 搜索框效果 | 搜索框的视觉效果 | 透明、模糊 |
| 模糊强度 | 模糊效果的强度（仅模糊效果时可用） | 1px - 30px |
| 搜索区域 | 是否显示整个搜索区域 | 开/关 |
| 纵向位置 | 搜索区域在页面中的上下位置 | 0% - 100%，默认 25% |
| 横向位置 | 搜索区域在页面中的左右位置 | 0% - 100% |
| 位置重置 | 快速将位置恢复默认 | 横向居中、纵向默认（25%） |
| 恢复默认 | 将所有设置恢复为默认值 | 点击按钮确认 |

## 常见问题

**Q: 安装后新标签页没有变化？**

A: 请确保在 `chrome://extensions/` 页面中，扩展已启用（开关为蓝色）。如果仍无效果，尝试移除扩展后重新加载。

**Q: 书签栏没有显示？**

A: 确保扩展已获得书签访问权限。在 `chrome://extensions/` 页面点击扩展详情，检查权限设置。

**Q: 背景图片不显示？**

A: 背景图片存储在浏览器本地存储中，如果清除浏览器数据可能会导致背景图片丢失，需要重新设置。

**Q: 如何卸载扩展？**

A: 在 `chrome://extensions/` 页面找到 "Vialike New Tab"，点击 "移除" 按钮即可。

## 技术说明

- 基于 Chrome Extension Manifest V3 开发
- 使用 Chrome Storage API 保存设置
- 使用 Chrome Bookmarks API 读取书签
- 支持图标缓存，提升加载速度
