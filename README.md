# Vialike New Tab - Chrome扩展

一个将Chrome新标签页替换为类似Via浏览器主页面的扩展，支持丰富的定制化选项。

## 功能特点

- **Logo定制**：自定义文字、字体大小、字体风格
- **搜索框定制**：圆角值、不透明度、描边粗细、描边不透明度、样式（矩形/线条）
- **背景定制**：背景图片、主题色亮暗、不透明度
- **高级选项**：搜索框效果（透明/模糊）、搜索区域显示/隐藏

## 安装步骤

### 第一步：生成图标

1. 用浏览器打开 `generate-icons.html` 文件
2. 点击"下载所有图标"按钮
3. 将下载的三个PNG文件（icon16.png、icon48.png、icon128.png）放入 `icons` 文件夹

### 第二步：加载扩展

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择 `vialike start page` 文件夹

### 第三步：使用

- 打开新标签页即可看到效果
- 点击右下角的设置按钮进行定制

## 文件结构

```
vialike start page/
├── manifest.json          # 扩展配置文件
├── newtab.html            # 新标签页HTML
├── styles.css             # 样式文件
├── newtab.js              # 主逻辑文件
├── generate-icons.html    # 图标生成工具
└── icons/                 # 图标文件夹
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 定制选项说明

### Logo设置
- **Logo文字**：显示在页面中央上方的文字，默认为"via"
- **字体大小**：24px - 120px
- **字体风格**：默认、Arial、Times New Roman、Georgia等多种选择

### 搜索框设置
- **样式**：矩形或线条两种风格
- **圆角值**：0 - 50px（仅矩形样式有效）
- **不透明度**：0% - 100%
- **描边粗细**：0 - 5px
- **描边不透明度**：0% - 100%

### 背景设置
- **背景图片**：可选择本地图片作为背景
- **主题色**：亮色或暗色模式
- **不透明度**：0% - 100%

### 高级设置
- **搜索框效果**：透明或模糊效果
- **搜索区域**：显示或隐藏整个搜索区域

## 搜索功能

- 输入网址直接跳转（支持带或不带 http:// 前缀）
- 输入搜索内容使用Google搜索
- 按Enter键或点击搜索按钮执行搜索
- 打开新标签页自动聚焦到搜索框
